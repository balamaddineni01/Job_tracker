All data is already in your Google Sheet — this CSV is a local backup.

1

Open your Google Sheet

Go to sheets.new to create a new spreadsheet. Name it Job Application Tracker.

2

Open Apps Script

In the sheet, click Extensions → Apps Script. Delete any code already in the editor.

3

Paste the backend code

Download Code.gs from your GitHub repo (it was included with this app). Paste the entire contents into the Apps Script editor. Click Save (floppy disk icon).

4

Deploy as a Web App

Click Deploy → New deployment. Set type to Web app.

Set Execute as: Me and Who has access: Anyone. Click Deploy and authorize when prompted.

5

Copy the Web App URL

After deploying, Google gives you a URL like:

https://script.google.com/macros/s/AKfycb.../exec

Copy that URL.

6

Paste the URL into this app

Close this tab, scroll up to the yellow banner, paste your URL and click Connect & sync. Your apps will start syncing immediately.

The URL is saved in your browser so you only need to do this once per device.

7

Using on a new device?

Just open your GitHub Pages URL, go to How to connect, and paste the same Apps Script URL. Your full history will load from the sheet automatically.
