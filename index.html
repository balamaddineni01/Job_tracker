Export data

Export all your applications as a CSV file to paste into Google Sheets.

Step 1 — Create your Google Sheet

Go to sheets.new and create a new spreadsheet. Name it Job Application Tracker.

Step 2 — Column headers (paste into row 1)

Put these in cells A1 through K1:

Date | Company | Role | Recruiter Email | Location | Portal | Status | Interview Date | Salary Range | Job Description | Notes

Step 3 — Freeze row 1 + enable filters

Select row 1 → View → Freeze → 1 row.

Then Data → Create a filter. Now you can instantly filter by status, company, portal, etc.

Step 4 — Add a Summary tab with formulas

Add a second sheet called Summary and use these formulas:

=COUNTIFS(Sheet1!A:A,">="&TODAY()) — today's applications

=COUNTIFS(Sheet1!A:A,">="&(TODAY()-WEEKDAY(TODAY(),2)+1)) — this week

=COUNTIFS(Sheet1!G:G,"Interview") — total interviews

Step 5 — Import your data

Go to the Stats tab and click Download CSV. Then in Google Sheets: File → Import → Upload the CSV → Append to current sheet.
